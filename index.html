UPDATE FIRMWARE

Enter Update Mode

  1. Power on HiChord
  2. Press and HOLD all three menu buttons: F1 + F2 + F3 for 5 SECONDS
  3. HiChord display shows: "Entering DFU Mode"
  4. Connect HiChord via USB-C to computer
  5. Click "Connect" button below
  6. Browser popup window appears
  7. Select "Daisy Bootloader" from device list
  8. Click "Connect" in popup window
  9. Click "Update Firmware" button below

⚠️ IMPORTANT

Popup must show "Daisy Bootloader" (NOT "HiChord" or "HiChord - Paired"). If incorrect device appears: Disconnect USB, repeat F1 + F2 + F3 hold for full 5 seconds.

Troubleshooting

Check Browser

Make sure you're using Google Chrome as your web browser.

While users have reported intermittent success with Firefox and Safari, Google Chrome is the only officially supported browser for flashing firmware to HiChord.

Reset USB Driver with ZadigWINDOWS ONLY

For Windows users, you may need to use Zadig to reset your USB driver.

  1. Download Zadig (use the most recent version)
  2. Launch Zadig
  3. Put HiChord into bootloader mode:
    • Power on HiChord
    • Press and HOLD all three menu buttons: F1 + F2 + F3 for 5 SECONDS
    • HiChord display shows: "Entering DFU Mode"
    • Connect HiChord via USB-C to computer
  4. In Zadig, click "Options" menu and select "List All Devices"
  5. In the dropdown menu, select "DFU in FS mode"
  6. In the target driver box (right side), select "WinUSB"

    Note: Do not worry if the box on the left displays "(NONE)" instead of "libusb0".

  7. Click "Replace Driver". Zadig will update to the appropriate driver to flash HiChord.

⚠️ BATCH 1 WARNING (KICKSTARTER LAUNCH EDITION)

If your HiChord is from Batch 1 (Kickstarter Launch Edition), make sure HiChord is fully charged before updating.
